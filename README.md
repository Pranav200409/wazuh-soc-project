# Wazuh SOC Project

## 📌 Overview
This project demonstrates a Security Operations Center (SOC) setup using Wazuh for threat detection and monitoring.

## 🔧 Features
- File Integrity Monitoring (FIM)
- SSH Brute Force Detection
- Threat Simulation (Ransomware-like behavior)
- MITRE ATT&CK Mapping
- Dashboard Visualization

## 🛠️ Tools Used
- Wazuh
- Kali Linux
- Ubuntu Server

## 📊 Project Workflow
1. Setup Wazuh Manager & Agent
2. Implement File Integrity Monitoring
3. Simulate SSH attacks
4. Perform threat simulation
5. Analyze alerts using dashboard

## 🔐 Attack Simulation
- SSH brute-force attack
- File deletion and modification (ransomware simulation)

## 📈 Outcome
Wazuh successfully detected and visualized security events in real-time.
